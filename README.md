# first-pr-practice

A practice repository for learning the GitHub pull request workflow.

## What this repo is for

This repo exists to practice opening, reviewing, and merging pull requests — the core collaboration loop on GitHub.

## How to contribute

1. Fork or clone the repo
2. Create a new branch: `git checkout -b my-change`
3. Make a small change and commit it
4. Push and open a pull request
